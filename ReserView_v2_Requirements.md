# ReserView v2.0 要件定義書

**プロジェクト名**: ReserView（リザビュー）- 予約管理・経営分析システム  
**バージョン**: 2.0  
**作成日**: 2025年12月2日  
**最終更新**: 2025年12月2日  

---

## 1. プロジェクト概要

### 1.1 目的
小規模店舗（治療院等）の経営者・経理担当者向けに、予約データを活用した経営管理ツールを提供する。日常の経理業務から経営分析、キャッシュフロー管理までを1つのアプリケーションで完結させる。

### 1.2 ターゲットユーザー

| ユーザー種別 | 利用モード | 主な用途 |
|-------------|-----------|----------|
| 経理スタッフ | 経理モード | 日々の売上確認、ステータス管理、金額修正 |
| 経営者 | 分析モード | リピート率確認、離反顧客管理、売上トレンド分析 |
| 経営者 | CFモード | 月次キャッシュフロー確認、資金繰り把握 |

### 1.3 現行システム（v1.0）からの変更点

- 3モード切替機能の追加
- 顧客分析機能（リピート率・離反率）の追加
- 売上トレンド分析機能の追加
- キャッシュフロー可視化機能の追加
- 経費データ管理機能の追加

### 3.4 StaffConfig シート（稼働率用）

分析モード「稼働率」で **スタッフ別メインレーン稼働率** を計算するためのマスタ。スプレッドシートに `StaffConfig` シートを作成し、1行につき1スタッフ（有効な先頭行が採用される）。

| 列名 | 型 | 説明 | 例 |
|------|-----|------|-----|
| 担当 | String | Reservations の `担当` と完全一致 | 佐藤 |
| メインレーン | Number/String | Reservations の `レーン` と一致（数値は正規化して比較） | 1 |
| 勤務開始 | Time | 勤務開始時刻 | 9:00 |
| 勤務終了 | Time | 勤務終了時刻 | 17:00 |
| 勤務曜日 | String | 月〜土の略字。区切りはカンマ・読点・空白等可 | 月,火,木,金 |
| 有効 | Boolean/String | 省略時は有効。`いいえ`/`無効`/`false`/`0` で無効 | はい |

**稼働率の定義（実装）**

- **分母**: 対象月の「月〜土の各日」について、StaffConfig の勤務曜日に該当し、かつ店舗営業枠（9:00〜18:00）と勤務時間の積集合が正となる分を合計した提供可能時間（分）。
- **分子**: Reservations のうち、キャンセル以外で、当該スタッフの `担当` が一致し、`レーン` が `メインレーン` と一致する予約の占有時間（分）。同一日内の重複はマージして二重計上しない。予約区間は店舗営業枠および当日の勤務枠の積にクリップする。
- **レーン列が無い** 場合: メインレーンによる絞り込みは行わず、担当のみで集計する。

### 4.3 分析モード（Analysis Mode）

**目的**: 経営判断に必要なデータを可視化・分析する

#### 4.3.1 サブタブ構成

```
[ダッシュボード] [顧客分析] [売上トレンド] [メニュー分析] [稼働率]
```

#### 4.3.2 ダッシュボード

**KPIカード表示**:

| 指標 | 計算方法 | 表示形式 |
|------|----------|----------|
| 今月売上 | 当月の確定売上合計 | ¥XXX,XXX |
| 前月比 | (今月 - 前月) / 前月 × 100 | +X.X% or -X.X% |
| 今月来院数 | 当月のユニーク患者数 | XX名 |
| リピート率 | 2回目以上来院者 / 全来院者 | XX.X% |
| 離反予備軍 | 3-6ヶ月未来院の患者数 | XX名（アラート色） |
| 離反確定 | 6ヶ月以上未来院の患者数 | XX名（警告色） |

#### 4.3.3 顧客分析タブ

**A. リピート率表示**

| 指標名 | 定義 | 計算式 |
|--------|------|--------|
| 期間内リピート率 | 選択月の来院者のうち過去にも来院履歴がある人の割合 | リピーター数 / 当月来院者数 × 100 |
| 新規定着率 | 初回来院から3ヶ月以内に2回目来院した人の割合 | 2回目来院者 / 新規患者数 × 100 |

**B. 離反管理**

| ステータス | 定義 | 表示色 |
|-----------|------|--------|
| アクティブ | 最終来院から90日未満 | 緑 |
| 離反予備軍 | 最終来院から90日以上180日未満 | 黄 |
| 離反確定 | 最終来院から180日以上 | 赤 |

**離反リスト表示項目**:
- 患者名
- 最終来院日
- 未来院日数
- 累計来院回数
- 累計売上
- よく利用したメニュー

**C. 来院回数分布**

棒グラフ表示:
- 1回のみ
- 2-5回
- 6-10回
- 11-20回
- 21回以上

#### 4.3.4 売上トレンドタブ

**A. 月別売上推移**

- グラフ形式: 棒グラフ + 折れ線（件数）
- 表示期間: 直近12ヶ月
- 表示項目: 売上金額、来院件数

**B. 曜日別分析**

- グラフ形式: 棒グラフ
- 表示項目: 曜日ごとの平均売上・平均件数

**C. 時間帯別分析**

- グラフ形式: ヒートマップ or 棒グラフ
- 時間区分: 9時台、10時台、...、20時台
- 表示項目: 時間帯ごとの予約件数

**D. 季節性分析**（データが2年以上蓄積後）

- 前年同月比の表示
- 季節トレンドのハイライト

#### 4.3.5 メニュー分析タブ

**A. メニュー別売上構成**

- グラフ形式: 円グラフ or ドーナツチャート
- 表示項目: メニュー名、売上金額、構成比

**B. メニュー別ランキング**

テーブル形式:
| 順位 | メニュー名 | 件数 | 売上 | 平均単価 |
|------|-----------|------|------|----------|

**C. メニュー別推移**

- グラフ形式: 積み上げ棒グラフ or 折れ線
- 選択したメニューの月別推移

#### 4.3.6 稼働率タブ

- **表示条件**: グローバル月が「年単位」ではなく特定の月を選択しているときのみ表示（年表示時は案内メッセージ）。
- **スタッフ選択**: `StaffConfig` に登録された担当一覧から選択。初期選択は先頭行。
- **KPI**: 月平均稼働率（メインレーン）、勤務日数、提供可能時間、最高/最低の曜日・時間帯。
- **グラフ**: 曜日別・時間帯別の稼働率（いずれも勤務枠を分母とした割合）。
- **API**: `getOccupancyStaffList()`（一覧）、`getOccupancyAnalysis(yearMonth, staffName)`（集計）。

#### 4.4.2 ブロックパズル図の構成要素

```
【お金のブロックパズル】

┌──────────┬─────────────────────────────────────┐
│          │            変動費                   │
│          ├───────────────────┬─────────────────┤
│          │                   │     人件費      │
│  売上高  │      粗利         │   （労働分配率）│
│          │                   ├─────────────────┤
│          │                   │  その他固定費   │
│          ├───────────────────┴─────────────────┤
│          │              利益                   │
└──────────┴─────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────┐
│ 税引後利益 │ 減価償却費 │ 返済 │ 設備投資 │繰越金│
└─────────────────────────────────────────────────┘
```

#### 4.4.3 計算ロジック

```javascript
// 基本計算
粗利 = 売上高 - 変動費
粗利率 = 粗利 / 売上高 × 100
固定費 = 人件費 + その他固定費
労働分配率 = 人件費 / 粗利 × 100
利益（営業利益） = 粗利 - 固定費

// キャッシュフロー計算
法人税等 = 利益 × 0.3（簡易計算、利益がマイナスなら0）
税引後利益 = 利益 - 法人税等
手元資金 = 税引後利益 + 減価償却費
繰越金 = 手元資金 - 借入返済 - 設備投資
```

#### 4.4.4 機能詳細

| 機能 | 説明 |
|------|------|
| 売上自動取得 | 選択月のReservationsから自動集計 |
| 前月コピー | 前月の経費データを複製して入力欄にセット |
| リアルタイム計算 | 入力値変更時に即座にブロックパズル更新 |
| 保存 | Expensesシートに書き込み |
| 履歴表示 | 過去の月別データをテーブル表示 |
| 推移グラフ | 月別の利益・繰越金推移を折れ線グラフ表示 |

#### 4.4.5 バリデーション

| 項目 | ルール |
|------|--------|
| 金額入力 | 0以上の整数のみ |
| 必須項目 | 売上高以外は0でも可 |
| 上限チェック | 売上高を超える変動費は警告表示 |

## 9. 用語定義

| 用語 | 定義 |

| 新規患者 | 初めて来院した患者 |

| 労働分配率 | 人件費 / 粗利 × 100 |
